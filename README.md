# 🎓 Student Data Management System

A full-stack web application for managing student records with authentication and CRUD operations.

## 🚀 Features

- Student Registration
- Login Authentication
- Add Student
- Update Student
- Delete Student
- Protected Routes
- REST API Integration


### Frontend

- React.js
- CSS3
- Axios

### Backend

- Node.js
- Express.js

### Database

- MongoDB

### Authentication

- JWT


## Installation

### Clone Repository

```bash
git clone https://github.com/swaraj-anjane/Student-Data-Management.git
```

### Backend

```bash
cd backend

npm install

npm run dev
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

## Environment Variables

```env
MONGO_URI=your_mongodb_uri

JWT_SECRET=your_secret

PORT=5000
```
